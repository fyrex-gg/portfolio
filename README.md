# Portfolio

Personal portfolio website showcasing my work and skills.

## Setup

This project uses Claude Code with Playwright MCP for development.

### Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development

## Project Structure

```
portfolio/
├── .claude/              # Claude Code configuration
│   ├── commands/        # Custom slash commands
│   └── context/         # Design principles and style guide
├── .mcp.json           # MCP server configuration
└── README.md           # This file
```

## Technologies

- HTML5
- CSS3
- JavaScript
- Playwright MCP (for testing and automation)

## Development

Built with Claude Code for AI-assisted development.
